E-mail contact

Note: If you need to get in touch with the developers of a particular project hosted here, please check the project's page (e.g. /projects/myprojecname) to find out the appropriate way to contact them.

For general help with using the service, you can subscribe to the users mailing lists at http://lists.gnu.org/mailman/listinfo/savannah-users and send mail to savannah-users@gnu.org (mails from non-members are accepted but moderated on a daily basis).

If you are the developer of one of the projects hosted here and you want to get in touch with the maintainers of this server (Savannah) you can then contact the Savannah Hackers by submitting a [support request] or sending mail to savannah-help-public@gnu.org (after checking the archives at http://lists.gnu.org/archive/html/savannah-hackers/) - when you want to contact us about security or confidential issues, you may also use the savannah-help-private@gnu.org list.

If you have found a bug in the Savane software (the software used to run this server), or you would like to request a new feature, you should use the Bugs system in Project Savane.
